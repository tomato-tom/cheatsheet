# Windows 10 ターミナル


ウィンドウ分割
`alt + shift + d`: ウィンドウの複製
`alt + shift + <方向キー>`: ウィンドウの仕切りをずらす
`alt + <方向キー>`: フォーカス移動

ウィンドウ
`ctrl + shift + n`: 新しいウィンドウ
`ctrl + shift + w`: 閉じる

タブ
`ctrl + alt + <n>: タブ<n>に切り替え
`ctrl + shift + d`: タブを複製
`ctrl + shift + t`: 新しいタブ
`ctrl + tab`: 次のタブ

スクロール
`ctrl + shift + pgup`: 1ページ上にスクロール
`ctrl + shift + pgdn`: 1ページ下にスクロール
`ctrl + shift + <方向キー>`: スクロール

編集
`ctrl + shift + a`: すべてのテキストを選択
`ctrl + shift + c`: コピー
`ctrl + shift + v`: 貼り付け
`ctrl + shift + f`: 検索


フォントサイズ
`ctrl + <plus>`: 拡大
`ctrl + <minus>`: 縮小
`ctrl + 0`: リセット
